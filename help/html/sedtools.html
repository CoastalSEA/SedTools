Requirements

SedTools is written in Matlab™ and requires v2016b, or later. In addition, SedTools requires both the dstoolbox and the muitoolbox.

Background

Utility to analyse settling column data based on the analysis method developed by Urs Neumeier, 2005. The source code of the core program and further details see web site.

SedTools classes

  • SedTools - defines the behaviour of the main UI.
  • SettlingParams - defines the input parameters.
  • SettlingAnalysis - handles the running of settling_column function, saving of results and display on the Q-Plot tab.

SedTools functions

settling_column - convert the record from the settling columns to grain size in units of mm or phi.
